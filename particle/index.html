Инфо:

развлекательная симуляция силы притяжения частиц с помощью javaScript - сила притяжения зависит от радиуса частицы, чем меньше частица, тем хуже она притягивается - точки появляются при нажатии на любую кнопку мыши

ВСЕ ПАРАМЕТРЫ МОЖНО (И НУЖНО МЕНЯТЬ) В МЕНЮ.

- можно пробовать ставить дробные значения (например, 0.01). - можно пробовать ставить отрицательные значения. - если страница зависнит, значит какое-то значение вызвало ошибку в скрипте, нужно просто обновить страницу - сила притяжения зависит от радиуса частиц, если на экране есть крупные частицы, при больших значениях частицы могут улететь, из-за большого ускорения, хотя при мелких частицах, от 0 до 1, большие значения не выкенут их за орбиты
